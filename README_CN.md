## 这是一个免费的开源的Minecraft安卓bedrock的作弊工具
- ### 构建步骤
  - 1. 克隆仓库
  ```Shell
      git clone https://github.com/SMlc666/MinecraftCheat.git
  ```
  - 2. 安装xmake
  ```Shell
      curl -fsSL https://xmake.io/shget.text | bash -s v2.9.6
  ```
  - 3. 编译项目
  ```Shell
    cd MinecraftCheat
    xmake
  ```
